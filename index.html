My Projects,

Sparky

Sparky is a game I am working on as a part-time developer at the company LungGames. This serious game was created to support young lung patients in doing lung function tests as they can be hard and seen as intimidating by young children. Sparky uses the concept of gamification to make these exercises more fun and engaging whilst also generating accurate results that can be easily read back online. The end goal of this game is to deploy it at hospitals as a tool to help young children with performing lung function tests.

GTRacing Pro

GTRacing pro is a time trial racing game made in Unreal engine. The goal of this game is to set the fastest lap time around the track. The track layout is inspired by the circuit Spa-Francorchamps in Belgium. This project was created in the span of 4 weeks and features an arcade style racing engine and controller rumble engine.

Space Shooter OSC

Space Shooter OSC is a space shooter game where you have to beat your opponent locally by using your Phone as a controller with the OSC (Open sound control) protocol. The OSC Protocol is powered by the OpenFrameworks library and the 3D Rendering is fully written by me using OpenGL Legacy.

The Cliff

The cliff is a VR platforming game made in 4 weeks in collaboration with several game artists. The goal of the game is to parkour to the other side of the cliff without falling off. You achieve this by jumping over platforms that appear and dissapear on a music beat. You also have ladders that you have to climb to finish.
